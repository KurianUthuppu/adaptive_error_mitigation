# Adaptive Error Mitigation (ibm_qamp_aem_15)

Repo for Adaptive Error Mitigation project in QAMP-2025.

## Overview

This repository provides tools and analytics for adaptive error mitigation in quantum computing, focusing on Qiskit-based workflows. It includes modules for extracting circuit features, analyzing layout mappings, characterizing backend noise, and computing derived noise metrics to identify and mitigate error hotspots in quantum circuits.

## Directory Structure

```
adaptive_error_mitigation/
    analytics/

    primitives/

    utils/
```

## Installation

1. Clone the repository.
2. Install dependencies:

```
   pip install -r requirements.txt
```

3. (Optional) Use the provided Python virtual environment in `.qcenv/`.
